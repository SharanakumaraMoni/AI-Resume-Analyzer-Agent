# 🤖 AI Resume Analyzer Agent

## 📌 Overview

AI Resume Analyzer Agent is an intelligent web application that analyzes resumes and provides feedback on skills, keywords, and ATS compatibility. It uses LLMs and NLP techniques to generate personalized suggestions for improving resumes.

---

## 🚀 Features

* 📄 Resume upload and text extraction
* 🧠 AI-powered resume analysis
* 🎯 ATS score and keyword matching
* 💡 Suggestions for improvement
* ⚡ Fast and interactive UI

---

## 🏗️ Architecture

User → Upload Resume → Text Extraction → LLM Analysis → Feedback Generation → UI Display

---

## 🛠️ Tech Stack

* Python
* LLM (Groq / Llama)
* LangChain / CrewAI
* FAISS (Vector Database)
* Streamlit / FastAPI

---

## ▶️ How to Run

```bash
git clone https://github.com/SharanakumaraMoni/AI-Resume-Analyzer-Agent.git
cd AI-Resume-Analyzer-Agent
pip install -r requirements.txt
streamlit run app.py
```
